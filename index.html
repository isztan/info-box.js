Info-Box.js

Info boxes are popovers that provide information about a specific item, it can load data from a specific url or from a given text or html. info-box.js is a jQuery plugin that allows show a info box popover easily.

You can download the info-box.js project from GitHub.

Examples

How to use

$('#item').infoBox({url:'/popovers/example.html #example-item'});

Options

Name Type Default Description
url string null URL to fetch the content of info-box. The url can contain the selector of the item you want to import from that page. http://awesomeurlexample.com #example-item
html string null If no url is specified, you can pass in the html or plain text you want to display in the info-box.
add_icon boolean false Adds an icon to the given element. If set to false the element needs to have a child or sibling element with the class name of info-box-icon
icon_image string null The name of the image of the icon that will be added.
icon_text string null If you don't want to use an image for the icon you can provide a text value instead. It will generate a link for the given url settings.
icon_in_element boolean false Indicates if the icon is inside of the given element or not. If set to true the icon will be a child element if set to false the icon will be a sibling element.
position string top Indicates where the info-box will be placed in relation to the icon. Possible values: top, bottom, left, right
width number 200 The pixel width of the info-box content.
offset number 3 The pixel size of the gap between the icon and the toolip.
arrow_size number 7 The pixel size of info-box popover arrow, it should be the same value as the one in the style.
top_bottom_arrow_offset number 30 This is the number in pixels that is set on the LESS/CSS file for the top and bottom triangles. It indicates how far from the left will the triangle be placed.
always_show boolean false If set to true the info-box will appear on load and will not disappear on click.
